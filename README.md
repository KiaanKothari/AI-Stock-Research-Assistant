# 📈 AI Stock Research Assistant

An AI-powered stock research platform built with **Python**, **Streamlit**, **Yahoo Finance**, and **Plotly**. This application enables users to explore real-time stock information, visualize historical price trends, and access key company metrics through an intuitive web interface.

This project is part of my growing portfolio in **Artificial Intelligence**, **Finance**, and **Quantitative Research**.

---

## ✨ Features

- 📈 Live stock market data
- 🏢 Company information and business overview
- 💰 Current stock price
- 🌍 Market capitalization
- 📊 Price-to-Earnings (P/E) ratio
- 📉 Interactive 6-month stock price chart
- 📅 52-week high and low
- 💵 Dividend yield
- ⚡ Fast and responsive Streamlit interface

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/KiaanKothari/AI-Stock-Research-Assistant.git
```

Navigate into the project directory:

```bash
cd AI-Stock-Research-Assistant
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
